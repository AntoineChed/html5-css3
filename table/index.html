Mise en Forme d'un Tableau

Structure HTML

Balises

  • table : tableau
  • caption : titre du tableau
  • thead : En-tête du tableau
  • tbody : Contenu du tableau
  • tr : Ligne d'un tableau
  • th : Cellule titre d'un tableau
  • td : Cellule d'un tableau

Attributs

  • colspan : défini le nombre de colonnes à fusionner
  • rowspan : défini le nombre de lignes à fusionner
  • scope : défini les cellules td auxquelles la cellule th est liée (row / col)
