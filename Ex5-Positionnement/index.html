Positionnement des élements en CSS

Le positionnement des élements d'une page HTML n'est pas chose aisée. CSS fournit un ensemble de méthodes qu'il est possible d'utiliser de manière cumulée :

  • Marge et espacement : via les propriétés margin et padding
  • Centrage : vertical ou horizontal via les propriétés text-align et vertical-align
  • Type d'Affichage : propriété display
  • Positionnement : propriété position
  • Objet Flottant : propriété float

1. Marge et Espacement

Les marges et Espacement sont à utiliser en priorité pour positionner un bloc par rapport à un autre ou par rapport à son bloc parent.
La propriété margin permettent d'appliquer un espacement extérieur au bloc, la propriété padding applique un espacement intérieur au bloc.

Paragraphe de texte avec un espacement intérieur de 5 pixels tout autour du texte (bordure grise en pointillée) englobé dans un bloc parent (bordure bleue) et d'un espacement extérieur de 15 pixels verical et 10 pixels horizontal. 

<div style="border: 2px solid blue;">
    <p style="padding: 5px; margin: 15px 10px; border:1px dashed gray;">
        Paragraphe de texte avec un espacement intérieur de 5 pixels tout autour du texte (bordure grise en pointillée) englobé dans un 
        bloc parent (bordure bleue) et d'un espacement extérieur de 15 pixels verical et 10 pixels horizontal
    </p>
</div>

2. Centrage Horizontal et Vertical

2.1 Centrage Horizontal

Pour centrer horizontalement un texte, on utilisera la propriété text-align. Pour centrer horizontalement un bloc, une des méthodes est d'utiliser la technique des marges automatiques (margin: auto - cf.ci-dessus). Une autre est de passer par des boîtes flexibles avec la propriété justify-content

Paragraphe de texte faisant 50% de la largeur du bloc parent, centré horizontalement et dont le texte est également centré horizontalement. 

<div style="border: 2px solid blue;">
    <p style="text-align: center; width:50%; margin: auto; border:1px dashed gray;">
        Paragraphe de texte faisant 50% de la largeur du bloc parent, centré horizontalement et dont le texte est également centré
        horizontalement.
    </p>
</div>

2.2 Centrage Vertical

Le centrage vertical va dépendre de l'élement conteneur.
Si le bloc conteneur dispose de la propriété display: table-cell comme une tableau par exemple, vous pouvez utiliser vertical-align: middle Sinon vous pouvez utiliser les boites flexibles et la propriété align-items

Paragraphe de texte de 50% de large, centré verticalement et horizontalement dans son bloc parent. 

<div style="border: 2px solid blue; height: 250px; width: 500px; display: flex; align-items: center; justify-content: center;">
    <p style="text-align: center; width:50%; border:1px gray dashed;">
        Paragraphe de texte de 50% de large, centré verticalement et horizontalement dans son bloc parent.
    </p>
</div>

3. Type d'Affichage

Changer le mode de rendu des élements HTML via la propriété display permet de facilement positionner les élements les uns par rapport aux autres. Display Block / Inline - Inline-block - Table cell - Flex (space between)

3.1 Display Block / Inline

Par défaut, les principaux élements HTML utilisent les rendus block (div, p...) ou inline (a, span...). Il est possible de modifier le mode de rendu d'un élement pour changer son mode d'affichage.
Exemple : utiliser une liste à puce contenant des liens pour générer un menu horizontal

  • Accueil
  • Rubrique 1
  • Rubrique 2
  • Rubrique 3
  • Contact
<nav>
    <ul class="menu">
        <li><a href="#">Accueil</a></li>
        <li><a href="#">Rubrique 1</a></li>
        <li><a href="#">Rubrique 2</a></li>
        <li><a href="#">Rubrique 3</a></li>
        <li><a href="#">Contact</a></li>
    </ul>
</nav>

3.2 Display Inline-Block / Table-cell...

D'autres type de mode de rendu permettent par exemple de mixer l'affichage inline et l'affichage block : inline-block qui permet d'ajouter une largeur, une hauteur, des marges et espacements à un bloc inline.
Le mode table-cell permet d'appliquer des propriétés d'une tableau à un bloc, généralement employé pour centrer verticalement. 

<div>
    <label style="display: inline-block; width: 280px; color: green;">Le label inline-block de mon champ</label>
    <input type="email" name="email" placeholder="Votre email" />
</div>

3.3 Display Flex

Il faudrait plusieurs TP pour couvrir toutes les possibilités apportées par les boîtes flexibles. Ce mode de rendu moderne et très puissant permet de positionner facilement des élements entre eux.

<div style="display:flex; justify-content: space-between; border: 2px solid blue; height: 80px;">
    <div style="width:30%;margin:1.5%;border: 1px solid green"></div>
    <div style="width:30%;margin:1.5%;border: 1px solid pink"></div>
    <div style="width:30%;margin:1.5%;border: 1px solid gray"></div>
</div>

4. Type de Positionnement

Par défaut tous les élements du flux HTML sont positionnés en statique => position: static
Il est possible de positionner des élements en modifiant la valeur de la propriété position

4.1 Position Fixe

Le positionnement fixe permet de sortir du flux HTML un élement et de le placer à un endroit précis en jouant sur les propriétés top, bottom, right et left. C'est le cas par exemple du header de cette page qui est fixe et dont voici le style CSS : 

header.header-fix {
    height: 85px;
    width: 100%;
    position: fixed;
    top: 0;
    left: 0;
    background-color: #eee;
    border-bottom: 1px solid #ddd;
    box-shadow: 0px 3px 5px 0px rgba(0,0,0,0.75);
}

4.2 Position Relative

Le positionnement relatif conserve l'élement dans le flux mais permet de le décaler par rapport à sa position initiale via les propriétés left, right, top et bottom.

Ce bloc est décalé de 10 pixels vers le bas et la droite
<div style="display:flex; justify-content: space-between; border: 2px solid blue; height: 80px;">
    <div style="width:30%;margin:1.5%;border: 1px solid green"></div>
    <div style="width:30%;margin:1.5%;border: 1px solid pink;position: relative; top: 10px; left:10px;">Ce bloc est décalé de 10 pixels
    vers le bas et la droite</div>
    <div style="width:30%;margin:1.5%;border: 1px solid gray"></div>
</div>

4.3 Position Absolue

Le positionnement en absolu sort l'élement HTML du flux et le positionne par rapport au bloc englobant (ayant la propriété de positionnement relatif) en fonction des valeurs fournies par top, left, right et / ou bottom.

Ce bloc est décalé de 10 pixels vers le bas et la droite par rapport au bloc parent
<div style="display:flex; justify-content: space-between; border: 2px solid blue; height: 80px; position: relative;">
    <div style="width:30%;margin:1.5%;border: 1px solid green"></div>
    <div style="width:30%;margin:1.5%;border: 1px solid pink;position: relative; top: 10px; left:10px;">Ce bloc est décalé de 10 pixels
    vers le bas et la droite par rapport au bloc parent</div>
    <div style="width:30%;margin:1.5%;border: 1px solid gray"></div>
</div>
