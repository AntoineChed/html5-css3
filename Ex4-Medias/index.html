Balises et attributs

  • balise img : image
  • attribut src : source du fichier image
  • attribut width : Largeur
  • attribut height : Hauteur
  • attribut alt : texte alternatif
  • attribut title : titre infobulle

Divers

Pour un chargement plus rapide, il est préférable que l'image se trouve dans un répertoire du site Web plutôt que sur un autre site Internet

Le texte de alt est affiché si l'image n'est pas affichable (mauvais lien src, format pas reconnu...) ou pour les personnes malvoyantes employant un leteur d'écran

Image simple

L'intégration d'une image se fait par la balise img dont l'attirbut src pointe vers le fichier image.
Il est une bonne pratique de rajouter les attributs : height et width pour réserver l'espace avant affichage de l'image ainsi que l'attribut alt (pour contenu textuel alternatif) qui va permettre de décrire l'image. 

<img src="assets/images/html5.png" height="250" width="250" alt="Logo HTML5" >

Balises et attributs

  • balise figure : unité de contenu
  • balise figcaption : légende de l'unité de contenu

Divers

La balise figure ne s'utilise pas exclusivement avec des images mais peu aussi être employée avec des vidéos, de l'audio, des formules mathématiques...

Images avec Légende

Il existe 2 balises HTML pour ajouter une légende à une image : figure et figcaption.

<figure>
    <img src="assets/images/html5.png" height="250" width="250" alt="Logo HTML5" title="Présentation d'une image légendée">
    <figcaption>Le Logo officiel de HTML5 disponible sur Wikipedia </figcaption>
</figure>

Images décoratives CSS

Certaines images d'une page Web ne servent qu'à agrémenter l'aspect général du site sans apporter d'informations complémentaires au contenu textuel.
Dans ce cas, il est possible d'utiliser CSS pour ajouter des images décoratives en utilisation la propriété background.

Ceci est mon paragraphe contenant une information hautement intéressante.
J'y ai ajouté une jolie image de fond qui n'apporte plus plus d'information mais c'est beau ! 

background: url('../images/background-image.jpg');
background-repeat: no-repeat;

Images Vectorielles

A l'inverse des images classiques JPG, GIF, PNG... qui sont des assemblages de pixles de couleurs différentes, les images vectorielles sont un assemblage de lignes et de formes.
Cela a pour avantage d'obtenir des images qui ne pixellisent pas et qui peuvent s'ajuster à n'importe quel taille de conteneur. Le format de ce type d'image est SVG.

Fichier Audio

L'insertion d'un fichier audio s'effectue par la balise audio. De la même manière que la vidéo, il est possible de définir plusieurs fichiers sources au cas où le navigateur ne prenne pas en charge le format spécifié

<audio controls>
    <source src="assets/audio/viper.mp3" type="audio/mp3">
    <source src="assets/audio/viper.ogg" type="audio/ogg">
    <p>Votre navigateur ne prend pas en charge l'audio HTML5. Voici, à la place, un <a href="assets/audio/viper.mp3">lien </a>.</p>
</audio>

Fichier Video

L'insertion d'un fichier video s'effectue par la balise video. Il est impératif par soucis de compatibilité de spécifier plusieurs fichiers sources au cas où le navigateur ne prenne pas en charge le format spécifié.
Il existe de nombreuses attributs facultatifs pur la balise video : width, height, autoplay, muted, loop, poster...

<video controls poster="assets/images/poster.png">
    <source src="assets/video/trailer.ogg" type="video/ogg">
    <source src="assets/video/trailer.m4v" type="video/m4v">
    <p>Votre navigateur ne prend pas en charge les videos HTML5.</p>
</video>
