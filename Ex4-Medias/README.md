# Example 4 - Les medias : image, vidéo, iframe, audio... 
Cet exemple présente l'intégration en HTML de contenus non textuels tels que :
* Image simple
* Image avec légende
* Image décorative CSS
* Image vectorielle
* Image réactive
* Fichier audio
* Fichier Vidéo
* Iframes

Démo : http://www.antoinechedebois.com/html5-css3/Ex4-Medias/
